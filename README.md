# Dog Tales Portfolio

A beautiful, responsive dog photography portfolio website with Facebook and Twitter/Instagram social media links.

## Files Included

- `index.html` - Main HTML structure
- `style.css` - All styling and responsive design
- `script.js` - JavaScript for interactions
- `README.md` - This file

## Setup Instructions

### 1. Create Your Project Folder

Create a folder called `dog-portfolio` and add these files:

```
dog-portfolio/
├── index.html
├── style.css
├── script.js
└── images/
    ├── hero-background.jpg
    ├── dog1.jpg
    ├── dog2.jpg
    ├── dog3.jpg
    ├── dog4.jpg
    ├── dog5.jpg
    └── dog6.jpg
```

### 2. Prepare Your Images

**Hero Background:**
- Size: 1920x1080px (landscape)
- Format: JPG
- Name: `hero-background.jpg`

**Dog Gallery Images:**
- Size: 800x800px (square)
- Format: JPG
- Names: `dog1.jpg`, `dog2.jpg`, `dog3.jpg`, `dog4.jpg`, `dog5.jpg`, `dog6.jpg`

### 3. Update Social Media Links

In `index.html`, replace these URLs with your actual profile links:

```html
<a href="https://twitter.com/yourprofile" target="_blank">
<a href="https://facebook.com/yourprofile" target="_blank">
```

### 4. Customize Content

**Change the title:**
- Line 8: `<title>Dog Tales Portfolio</title>`
- Line 15: `<h1 class="site-title">Dog Tales</h1>`

**Update the hero text:**
- Line 32: Change the main title
- Lines 33-35: Update the subtitle description

**Change contact form language:**
If you want English instead of Spanish:
- Line 63: Change `CONTACTAME` to `CONTACT ME`
- Line 66: Change `Nombre` to `Name`
- Line 70: Change `Mensaje` to `Message`
- Line 73: Change `ENVIAR` to `SEND`

### 5. Preview Your Website

**Using Live Server (Recommended):**
1. Install "Live Server" extension in VS Code
2. Right-click `index.html`
3. Select "Open with Live Server"

**Manual Method:**
1. Double-click `index.html` to open in your browser
